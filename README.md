# APIS 
> ## To Get the Listings
> -  send Get Request to ``${Website_URL}/KuhainAngListing`` data type is in `json`

> ## Send New Listing
> - Send Post Request  to `${Website_URL}/BagongBenta` 
>   ### Parameters:
>   - `'title'`
>   - `'descriotion'` 
>   - `'price'`       must be Integer
>   - `'category'`    - Any Category
>   - `'link'`        - Image Link
>   ### Returns HTML
